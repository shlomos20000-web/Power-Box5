# Healthy Snack Box Landing Page

A modern, conversion-optimized landing page for the Healthy Snack Box (42 Count) product, built with React, TypeScript, Tailwind CSS, and Framer Motion.

## 🎯 Features

### 🚀 Modern 2025 Design

- **Glassmorphism**: Beautiful frosted glass effects with backdrop blur
- **Fresh Color Palette**: Emerald green primary with orange accents
- **Rounded Corners**: 2xl border radius for modern aesthetics
- **Micro-interactions**: Smooth hover effects and animations

### 📱 Fully Responsive

- **Mobile-first design**: Optimized for all screen sizes (360px to 1440px+)
- **Responsive typography**: Scales beautifully from mobile to desktop
- **Touch-friendly**: Optimized for mobile interactions

### 🎭 Animations & Interactions

- **Framer Motion**: Smooth page transitions and scroll animations
- **Hover Effects**: Interactive product cards and buttons
- **Scroll Animations**: Elements animate in as you scroll
- **Modal Animations**: Smooth open/close transitions

### 🛒 Conversion-Optimized Flow

1. **Hero Section**: Compelling headline with CTA to scroll to product
2. **Product Card**: Click opens modal (no direct purchase)
3. **Modal**: Detailed product info with "Buy on Walmart" button
4. **Final CTA**: Secondary conversion opportunity

### ♿ Accessibility Features

- **Keyboard Navigation**: Full tab navigation support
- **ARIA Labels**: Proper accessibility attributes
- **Focus Management**: Trapped focus in modal
- **Escape Key**: Close modal with ESC key
- **Color Contrast**: WCAG compliant color ratios

### ⚡ Performance Optimized

- **Lazy Loading**: Images load as needed
- **Optimized Assets**: WebP image format
- **DNS Prefetch**: Preload external domains
- **Bundle Optimization**: Code splitting and tree shaking

### 🔍 SEO Ready

- **Meta Tags**: Complete OpenGraph and Twitter Card setup
- **Structured Data**: JSON-LD schema for product information
- **Semantic HTML**: Proper heading hierarchy (H1, H2, H3)
- **Fast Loading**: Optimized for Core Web Vitals

## 🛍️ Product Information

- **Product**: Healthy Snack Box (42 Count) by Gift-A-Snack
- **Features**: 42 individually wrapped snacks, mix of sweet & savory
- **Target**: Perfect for work, school, and travel
- **Purchase**: Available on Walmart with secure checkout

## 🏗️ Tech Stack

- **React 18**: Modern React with hooks
- **TypeScript**: Full type safety
- **Vite**: Fast build tool and dev server
- **Tailwind CSS**: Utility-first styling
- **Framer Motion**: Animation library
- **Radix UI**: Accessible component primitives
- **Lucide React**: Beautiful icons

## 🚀 Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Type checking
pnpm typecheck

# Run tests
pnpm test
```
